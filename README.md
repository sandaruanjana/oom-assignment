# Simple Social Media Application

## Installation

Prerequisites:
- Netbeans IDE 18.0
- Java 8
- Maven

## Run Locally

Clone the project

```bash
  git clone https://github.com/sandaruanjana/oop-assignment.git
```

Go to the project directory

```bash
  cd oop-assignment
```

Run Maven

```bash
  mvn clean install
```

Then run the jar file

```bash
  java -jar target\social-jar-with-dependencies.jar
```
